# NotCVE Reports

Weekly Vulnerability Digest — published every Thursday from the
[notcve.org](https://notcve.org) corpus.

Every figure in these reports is generated **deterministically** from the
corpus and **machine-validated** before publication — nothing is estimated,
rounded creatively, or written from memory. The same data is searchable at
[notcve.org](https://notcve.org).
